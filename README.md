# 🎸 Guitar Scale Visualizer

Una aplicación web **Premium** e interactiva diseñada para músicos y estudiantes de guitarra que desean dominar el diapasón y la teoría musical de forma visual y auditiva.

## ✨ Características

- **🎸 Mástil de Guitarra Interactivo**: Visualiza cualquier escala en un mástil de 15 trastes con soporte para múltiples afinaciones (Standard, Drop D, DADGAD, etc.).
- **🎹 Teclado de Piano Sincronizado**: Las notas seleccionadas se reflejan automáticamente en un teclado de piano de 2 octavas.
- **🔊 Sonido Real (Tone.js)**: Haz clic en cualquier nota del mástil o tecla del piano para escuchar su tono real con síntesis polifónica.
- **🎨 Estética Premium Dark**: Interfaz moderna con modo oscuro, acentos naranja-oro, tipografía "Outfit" y efectos de cristal (Glassmorphism).
- **🎼 Biblioteca de Escalas**: Soporte para escalas Mayores, Menores, Pentatónicas, Blues y Modos Griegos.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: [React 19](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **UI Framework**: [Material UI (MUI) v9](https://mui.com/)
- **Audio Engine**: [Tone.js](https://tonejs.github.io/)
- **Tipografía**: Google Fonts (Outfit)
- **Iconos**: Material Icons

## 📦 Instalación y Uso

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/javiboin/guitar-scales-react.git
   cd guitar-scales-react
   ```

2. **Instala las dependencias:**
   ```bash
   npm install
   ```

3. **Inicia el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construye para producción:**
   ```bash
   npm run build
   ```
